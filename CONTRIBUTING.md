# Contributing

Thanks for helping grow this educational resource!

## Adding a new project

1. **Check the license.** Only add projects that have a license permitting redistribution, or where the author has made the source publicly available. If the license is unclear, note it as `"unknown"` — we will not build binaries for projects with unclear licensing until clarified.

2. **Always link to the original repo.** Every entry must include a direct URL to the original source repository. This is non-negotiable.

3. **Respect the original author's wishes.** If a project's README or license contains restrictions on redistribution, aggregation, or automated building, do not add it here.

4. **Add to both files:**
   - `README.md` — Add an entry in the appropriate category table with a clickable link to the original repo.
   - `projects.json` — Add a build entry with the repo URL, build directory, and license.

## Takedown requests

If you are the author or rights holder of any project listed here and would like it removed:

- **Open an issue** titled "Takedown request: [project name]"
- Or contact us directly

We will remove the entry and any associated build artifacts **promptly, no questions asked**.

## Code of conduct

This is an educational resource. Be respectful of the original authors' work. Give credit. Follow their licenses. Link back to their repos.
